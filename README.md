# Mi_proyectoreflex
Pagina web de reflex

Este proyecto es una aplicación web básica creada con **Reflex** en Python.  
Incluye un título principal, un texto de bienvenida y un botón interactivo que cambia el estado al hacer clic.

##  Requisitos previos
- Python 3.10 o superior
- Entorno virtual (`venv`)
- Reflex instalado
